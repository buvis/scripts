ConfigAdapter documentation¶

class buvis.adapters.config.config.ConfigAdapter(file_path: Path | None = None)¶

Bases: object

Manages global runtime configuration for BUVIS scripts. Provides functionality to load from YAML file, access, and modify configuration settings.

If no configuration file path is provided, then it tries to find the path from the BUVIS_CONFIG_FILE environment variable. If the environment variable is not set, it defaults to ~/.config/buvis/config.yaml.
