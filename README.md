# BUVIS scripts

Standalone scripts for Bob's Universal and Very Intelligent System.

Feel free to reuse anything. If you spot improvements, email <tomas@buvis.net>.

## Prerequisites

- [uv](https://docs.astral.sh/uv/)

## Usage

Add `bin/` to your PATH. Scripts are self-bootstrapping via `uv run --script` (PEP 723 inline metadata).

```bash
bin/slugren my-file.txt
```
